# FindMaestro
Who is the real maestro?

This is a starter Unity project for the game *Find Maestro*.  
Currently it contains only an empty URP template with basic first-person movement (to be added later).

## Requirements

- Unity 2022.3 LTS
- Universal 3D (URP) support (included in the template)

## How to Run

1. **Clone the repository**  
   ```bash
   git clone https://github.com/xXwuyang/FindMaestro.git
2. **Open the project in Unity Hub**
   
   Click "Add project" and select the FindMaestro folder inside the cloned repository.
   Make sure the project uses Unity 2022.3 LTS.
3. **Open the main scene**
   
   In the Project window, navigate to Assets/Scenes/.
   Double-click SampleScene.unity (or whatever the default scene is named).
4. **Play**
   
   Press the Play button in the Unity Editor.
